Portifólio

Apresento aqui um projeto de banco de dados para um sistema de locação de veículos, com modelagem, dados e consultas SQL.

Modelagem e Dados

O sistema possui as seguintes tabelas principais:

  • Cliente: informações pessoais dos clientes.
  • Veículo: dados dos veículos disponíveis para locação.
  • Manutenção: registros de manutenções realizadas nos veículos.
  • Pagamento: informações sobre pagamentos realizados.
  • Locação: dados das locações feitas pelos clientes.
  • LocacaoVeiculo: relação entre locações e veículos alugados.

Consultas SQL

Exemplo de consulta para listar veículos disponíveis:

SELECT modelo, marca, ano, placa, valorDiaria
FROM Veiculo
WHERE estado = 'Disponível';

Exemplo de consulta para obter locações ativas:

SELECT L.idLocacao, C.nome AS Cliente, V.modelo AS Veículo, L.dataInicio, L.dataFim
FROM Locacao L
JOIN Cliente C ON L.idCliente = C.idCliente
JOIN LocacaoVeiculo LV ON L.idLocacao = LV.idLocacao
JOIN Veiculo V ON LV.idVeiculo = V.idVeiculo
WHERE L.dataFim >= CURRENT_DATE;
